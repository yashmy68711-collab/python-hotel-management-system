# Python Hotel Management System

## About

A CLI Hotel Management System built using Python. This application helps manage hotel rooms by adding rooms, booking rooms, checking out customers, and searching room details using Object-Oriented Programming.

---

## ✨ Features

✔️ Add new rooms
✔️ View all rooms
✔️ Book available rooms
✔️ Check out customers
✔️ Search room details
✔️ Room availability tracking
✔️ Input validation

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)
* Dictionaries
* CRUD Operations

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose an option from the menu
5. Manage hotel rooms and bookings
